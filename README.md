# solana-bundling-toolkit

This package exports a series of toolkits that you can use to create pool

## Installation

```bash
npm install solana-bundling
or
yarn add solana-bundling
```

## Example Usage

```js
if (USING_JITO) {
  createPoolAndInitialBuy();
} else {
  craetePool();
}
```
